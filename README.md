# League of legends profile API
## _Basic application that searches the profile of league of legends players_

Using the API provided by Riot Games, it is possible to consult the profile of the players in a very simple way.

- It is necessary to generate a Riot Token on the Riot Games developer portal
- Copy the Riot Token to the SummonerRestApi file, in the Header X-Riot-Token.
- use swagger to return the player's profile, enter the nick.
